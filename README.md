
### Usage

```
$ bscope
Cloning https://github.com/twbs/bootstrap...
Cloned.
Checking out from v4.1.3-87-g6e2a823...
Using scope name: .bootstrap-scope
Generating template...
Writing to bootstrap-scoped-6e2a823.gzip
```

That's basically it! Run the help for more details...

```
$ bscope --help
```
